# Event Report: “Mini Meetup 06/06”

### Purpose of the Event
- To stay updated on core virtualization technologies (Containerization) and Graph Database architectures.
- To analyze the integration of Cloud Security and Machine Learning for intrusion detection.
- To share practical experiences regarding career advancement in the IT industry and effective teamwork skills.
- To expand knowledge on network programming via WebSocket protocols.

### List of Speakers
- **Bảo Huỳnh** - Topic: Docker - A containerization technology
- **Lê Hoàng Gia Đại** - Topic: Combining AWS WAF with Machine Learning for Cyber Attack Detection on AWS
- **Việt Phát** - Topic: AWS Neptune for Building a Graph Knowledge Base for GraphRAG
- **Nguyễn Quốc Bảo** - Topic: Godot WebSocket
- **Vinh Trần** - Topic: From IT Helpdesk to Senior Sysadmin: Self-learning Journey and the Transition Roadmap to Cloud/DevOps
- **Trương Phước** - Topic: Effective Teamwork Skills

### Key Highlights

#### Cloud & Infrastructure
- **Docker Technology (Speaker: Bảo Huỳnh):** Clarified Containerization concepts, explaining how to package applications with their dependencies to definitively resolve the "it works on my machine" dilemma.
- **Network Programming (Speaker: Nguyễn Quốc Bảo):** Explored WebSocket protocols within Godot, demonstrating how to establish full-duplex, low-latency connections highly applicable for real-time systems.

#### AI & Advanced Security
- **Cyber Attack Prevention (Speaker: Lê Hoàng Gia Đại):** A breakthrough solution integrating AWS WAF with Machine Learning models. ML models analyze anomalous behaviors to proactively block Zero-day attacks much more effectively than static rule-based configurations.
- **Graph Databases (Speaker: Việt Phát):** Introduced AWS Neptune and GraphRAG architectures. Demonstrated using graph databases to construct Knowledge Bases, enabling AI models to retrieve highly interconnected and complex information accurately.

#### Career Orientation & Soft Skills
- **Cloud/DevOps Roadmap (Speaker: Vinh Trần):** An inspiring story and a practical roadmap charting the transition from an IT Helpdesk role to a Senior Sysadmin and Cloud/DevOps Engineer.
- **Teamwork Skills (Speaker: Trương Phước):** Methodologies for communication, conflict resolution, and cross-functional task management to optimize productivity within tech project teams.

### Lessons Learned

#### Architectural & Technical Mindset
- **Container-first:** Fully grasped the necessity of Dockerizing applications from the very beginning to ensure absolute consistency across environments (Dev, Staging, Production).
- **Proactive Security:** Shifted from a reactive security mindset to a proactive one by leveraging Machine Learning within WAF to analyze and predict threat vectors.
- **Graph-based Thinking:** Understood how organizing data in a graph format (Nodes and Edges), rather than traditional tables, highly optimizes AI tasks involving complex data relationship analysis.

#### Personal Development
- Acquired a clear learning roadmap to dive deeper into Cloud Operations and DevOps domains.
- Realized the paramount importance of "Soft skills" like communication and Agile/Scrum teamwork processes, which dictate the success or failure of real-world projects.

### Application to Work
- **PawVerse Infrastructure Deployment:** Immediately applied Bảo Huỳnh's Docker insights to write User Data scripts that automatically install and run containers on EC2 instances (Week 5 & Week 10).
- **Firewall Configuration:** Leveraged the operational principles of AWS WAF (from Mr. Đại's sharing) to configure the Edge Layer security perimeter in tandem with CloudFront.
- **Team Communication:** Proactively applied teamwork skills to communicate clearly and report progress transparently to my mentors (Mr. Nguyen Gia Hung) throughout the internship.

### Event Experience

Participating in the **FCAJ Tech Sharing** session provided me with multi-dimensional knowledge. Unlike purely theoretical lectures, this was a forum filled with experiences derived from blood, sweat, and tears.

#### Learning from Top-Tier Experts
- The selected topics were highly relevant to current industry demands. From core infrastructure (Docker) to advanced security (WAF + ML), the speakers successfully broke down complex issues into easily digestible concepts.

#### Practical Insights
- Hearing Vinh Trần's real-world career story was a massive morale boost. It helped me, an upcoming graduate, clearly visualize the path ahead, the certifications required, and the practical skills needed to become a true Systems Engineer.

#### Key Takeaways
- Technology shifts daily. Modern systems demand more than just basic computer networking knowledge; they require integrating AI and Machine Learning to harden security. Furthermore, teamwork skills and a continuous learning attitude are the ultimate keys to going far in this industry.
