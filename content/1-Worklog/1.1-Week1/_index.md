* Grasp the core concepts of Cloud Computing and AWS Global Infrastructure.

### Week 1 Achievements:

* **Foundation:** Understood AWS infrastructure design philosophy, specifically how Availability Zones (AZs) operate independently to ensure high availability.
* **Account Governance & Security:** Successfully established a secure account environment; recognized Root user risks, configured cost alerts (AWS Budgets), and learned troubleshooting via AWS Support.
* **Access Management:** Mastered IAM access control mechanisms. Installed and configured AWS CLI, understood Access Key security flows, and applied IAM Roles for EC2 instead of hard-coding sensitive credentials.
* **Practical Infrastructure Deployment:** Applied computer networking knowledge to manually configure virtualized network infrastructure (Amazon VPC) and seamlessly deploy application servers (Amazon EC2).
